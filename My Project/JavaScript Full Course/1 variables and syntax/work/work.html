Tarea: Operaciones con datos

Descripción de la tarea

En esta tarea, se me solicita crear un script en JavaScript que realice operaciones con datos ingresados por el usuario y muestre los resultados. Los pasos a seguir son:

  1. Saludo con nombre al usuario
  2. Suma de números
  3. Mostrar los resultados en la consola y en una alerta.
  4. Concatenación de textos

La conexión entre archivos

Para realizar la tarea, cree tres tipos de archivos:

  • Archivo HTML: work.html
  • Archivo JavaScript: work.js
  • Archivo CSS: styles.css

Todos los archivos están en la misma carpeta
